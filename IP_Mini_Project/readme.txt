To run the code, run the main MATLAB code Layer_Separation.m

The codes Intrinsic_Decomposition.m & Reflection_Removal.m are functions for the two 
layer separation techniques.

The code Reflectance_generate is used to make the image for Reflection Removal. 

The images used are:
Intrinsic Decomposition: intrinsic_cup.png
Reflection Removal: reflectance_subject_tampered.png
Reflection Generate: lena_colored_256.bmp & idk_256.bmp

To change the images, the image path can be changed in the codes Layer_Separation.m & Reflectance_generate.m
